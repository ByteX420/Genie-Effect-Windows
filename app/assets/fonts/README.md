# Inter

Bundled static faces from the official [Inter](https://github.com/rsms/inter) **v4.1** release
(`extras/ttf/`):

| File | Weight | Used for |
|------|--------|----------|
| `Inter-Regular.ttf` | Regular | Labels, helpers, buttons, combos, idle nav, values |
| `Inter-SemiBold.ttf` | SemiBold | Hero/section titles, selected nav, product name |
| `Inter-Bold.ttf` | Bold | Page titles only |
